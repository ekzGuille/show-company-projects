# POC Listado credenciales

## Objeto input *(json)*

* Empresa *`Object`*
  * Nombre *`String`*
  * Entornos *`Array`*
    * Nombre entorno *`String`*
    * Url entorno *`String`*
    * Usuario entorno *`String`*
    * Contraseña entorno *`String`*
    * SAP Logon ? *`Object`*
      * Usuario SAP Logon *`String`*
      * Contraseña SAP Logon *`String`*
    * VPN *`Object`*
      * Usuario VPN *`String`*
      * Contraseña VPN *`String`*
